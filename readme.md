## 开发

克隆git仓库，安装依赖项并启动应用程序

```bash
git clone 
npm install
npm run dev
```

注意：使用“depth”参数可以减小克隆大小并加快克隆速度。

```sh
git clone --depth=1 https://github.com/XXX.git
```
